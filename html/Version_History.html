Hytool - The versions

This document summarizes the history of the different versions of hytool.

Version 0.0

  • Start of development in 1998

Version 1.0

October 2001

  • The first documented version

Version 1.2

December 2002

  • Improvement of the Graphical reports (tst)
  • New solution : Gringarten and Ramey
  • Improvement of the step drawdown method (eha)
  • Improvement of the step drawdown method (eha)

Version 1.3

August 2003

  • Improvement of slug tests interpretations
  • Modification of all the functions related to the Cooper et al. solution (replacing alpha by Cd) and modification of the guess algorithm (gss) addition of a radius of investigation calculation
  • Implementation of the solution of Ramey et al. with a skin effect
  • Implementation of the solution of Dougherty with partal penetration (dls)
  • Creation of ldf_slg to enhance the efficiency of slug test interpretation

Version 1.4

October-December 2003

  • Improvement of the Barker grf model
  • Development and validation of the Gringarten model (grg)
  • Development of an interactive windowing for the Theis method and for slug test (ezwt and ezst)
  • Implementation of the Cinco-Ley solution for fractured wells
  • Implementation of the Hvorslev model for slug test
  • Implementation of the prs model for spherical flow slug test

Version 1.5

Jannuary-March 2004

    185 M-files, 2366 lines of code, 1000 lines of comments
  • Improvement of the Warren and Root technique
  • Updating the documentation
  • Implementation of the graphical interfaces ezwt and ezst

Version 1.5.2

April 2005

  • Corrections of the help files to make them homogeneous
  • Correction of eha_tst to replace the variable name C by B
  • Correction of eha_tst so that it accepts both a line vector and a column vector for p
  • Correction of thn_gss

Version 1.6

September 2005

  • Implementation of the solution of Delay et al. 2004: functions del_dim and del_tst
  • Implementation of the solution trm for Theis Multiple Rates tmr_pre, tmr_dim, etc.

Version 1.7

Mai 2006

  • Corrections of the help in tmr
  • Implementation of the htj_gss, fit, and tst functions
  • Modification of diagnostic(t,s) to open a single figure

Version 1.8

Novembre 2006

  • Correction d'un bug dans la solution de Warren et Root war_dls
  • Mise à jour de hmb_drw et pca_drw

Version 1.9

March 2007

  • Correction de GRF - Ajout des fonctions gss, fit et tst
  • Adding new function ths_dmo
  • Correction of Contents.m

Version 1.91

Mai 2007

  • Corrections of a few bugs
  • New functions jlq_dim, jlq_gss, jlq_fit, jlq_tst, jlq_dmo
  • New functions birsoy_time, birsoy_demo
  • Correction of Contents.m

Version 2.0beta

July 2007

  • Development of a set of new functions.
  • nzl Neuzil for pressurized test
  • tmc Multiple rate constant head
  • filters and samplers
  • hbw and hba Hamm and bidaux
  • pca and pcb Papadopulos cooper in aquifer
  • aga Agarwal in aquifer
  • Corrections of a few bugs in the figures created with tst

Version 2.0

October 2007

  • Release of the version 2.0

Version 2.01

April 2008

  • Online distribution of version 2.0 on sourceforge.net, followed by 2.01
  • Corrections of a some errors in the help files, and in the syntax of the commands hyselect, nsl_rpt, nsl_gss, csl_rpt, blt_pre

Version 2.02

October 2008

  • Modification of the fit() function in order to compute confidence interval and display the history of the fitting process.

Version 2.03

May 2010

  • Adding the WAW model for single well test in double porosity aquifer.
  • Minor bug corrections
