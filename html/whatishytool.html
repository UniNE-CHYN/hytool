Hydrogeological Toolbox

What is Hytool?

Hytool provides an open environement for the interpretation of hydraulic tests in matlab.

The toolbox contains some of the most usual analytical solutions used to describe groundwater flow around wells, as well as functions for importing, displaying the data, and fitting a model to the data.

One of the main interest of hytool is that it can be extended by the user to include additional solutions when they are required.

The user can also automate tasks and interpret series of data set rapidly.
